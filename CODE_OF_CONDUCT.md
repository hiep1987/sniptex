# Code of Conduct

SnipTeX adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be welcoming, assume good faith. Harassment, hateful conduct, or personal attacks are not tolerated in any project space (issues, pull requests, Discord, code reviews).

Report incidents privately to the maintainers via the contact email listed in `README.md`. Reports are handled confidentially.
